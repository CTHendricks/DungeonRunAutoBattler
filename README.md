# DungeonRunAutoBattler
Base Idea of Game:
 - Player starts with very simple, possibly different/random each time, starting party.
 - Each battle will net some kind of rewards (Normal = Gold, Boss = Items/Characters, Super Boss = Augments).
 - These rewards will come in the form of "Buckets" that will vary and give the player choices on build direction.
 - Augments will carry onward during the run possibly affecting entire battlefields of units.
